# 全唐诗查询与鉴赏 Skill（tang-poetry-explorer）

## 概述

本技能基于《全唐诗》数据库（收录唐代2200余位诗人、近5万首诗作），围绕"意象主题"进行深度检索、分析和鉴赏。用户输入一个意象主题（如梅花、竹、月、江水等），技能自动检索相关诗作及诗人，提供六维度深度分析。

## 关键词释义

> **意象**：唐诗中反复出现的、承载特定情感和文化内涵的物象，是诗人表达情感的载体。例如"梅"象征孤高，"月"寄托思乡，"雁"承载离别，"竹"代表气节。

### 核心能力

1. **意象主题检索** — 从全唐诗中检索所有涉及该意象的代表诗作及诗人
2. **诗人画像与比较** — 生平简介、创作风格、同题材风格差异对比
3. **逐首深度鉴赏** — 原文、注释、译文、逐句赏析、意境分析、艺术手法
4. **意境流派归类** — 同一意象按意境分类（如梅花可分为傲雪型、暗香型、折梅型、梅花落型）
5. **跨意象关联分析** — 意象之间的关联网络与组合规律
6. **学习路径推荐** — 入门级→进阶级→深度级三级递进学习路径

## 工作流程

### 阶段一：接收用户输入并解析意象主题

用户输入一个或多个意象主题，系统自动归类并展开检索。

**支持的意象类别：**

- **植物类**：梅、兰、竹、菊、松、柳、荷、桂、莲、桃、杏、枫
- **天象类**：月、日、星、云、雨、雪、风、霜、露、霞、虹
- **地理类**：山、水、江、河、湖、海、溪、泉、石、桥、亭、楼
- **动物类**：雁、鹤、蝉、莺、马、燕、鸥、鹭、鸳鸯、杜鹃、猿
- **季节类**：春、夏、秋、冬
- **情感/情境类**：离别、思乡、壮志、闲适、怀古、羁旅、闺怨、隐逸、边塞、田园
- **器物类**：琴、剑、酒、茶、灯、笛、箫、舟、窗、镜

**输入示例：**
- "梅花" — 单一意象查询
- "写竹子的诗" — 植物类意象
- "李白写月的诗" — 特定诗人的特定意象
- "秋天和大雁" — 复合意象查询
- "唐诗中的送别场景" — 情感/情境类查询

### 阶段二：多维联网检索

对意象主题执行至少4轮搜索，覆盖以下信息源：

#### 第一轮：权威古籍数据库（≥2轮）
- `site:ctext.org <意象> 唐诗` — 中国哲学书电子化计划
- `site:so.gushiwen.cn <意象>` — 古诗文网搜索
- `<意象> 全唐诗 中华书局` — 权威版本查询

#### 第二轮：学术赏析资源（≥1轮）
- `"唐诗三百首" <意象> 鉴赏`
- `<意象> 唐诗 名篇 赏析 意境`
- `site:zhihu.com <意象> 唐诗 分析` — 知乎学术讨论

#### 第三轮：诗人专题研究（≥1轮）
- `<诗人名> <意象> 风格 人生经历`
- `<诗人名> 诗歌特点 文学流派`
- `site:baike.baidu.com <诗人名>` — 百度百科诗人条目

#### 第四轮：比较研究与跨意象分析（≥1轮）
- `<意象> 唐诗 宋词 演变`
- `<意象A> <意象B> 组合 唐诗`
- `<意象> 意象分类 类型学`

#### 搜索执行规则
1. 每次查询至少执行4轮搜索
2. 搜索结果去重，保留信息增量最大的版本
3. 标注每个信息源的类型（古籍/学术/赏析/百科）
4. 诗文引用必须忠实于《全唐诗》原文，不可杜撰
5. 无法确认的信息明确标注"待查证"或"此处引用待核实"

### 阶段三：诗人画像与比较

对涉及该意象的代表诗人，逐一提供以下分析：

#### 3.1 生平简介
- **时代背景**：初唐（618-713）/ 盛唐（713-765）/ 中唐（765-835）/ 晚唐（835-907）
- **人生轨迹**：官职经历、重要人生转折、时代背景影响
- **文学流派**：山水田园派、边塞派、浪漫派、现实派、新乐府运动等

#### 3.2 意象偏好与创作风格
- 该诗人偏爱的意象群（如李白偏爱月、酒、剑、山水）
- 语言风格特征（豪放飘逸 / 沉郁顿挫 / 清新淡雅 / 绮丽华美 / 平易通俗）
- 核心艺术特色与独创之处

#### 3.3 同题材风格差异对比

对比框架（以"月"意象为例）：

- **李白写月** — 浪漫飘逸、超然物外。"举杯邀明月，对影成三人"——月是诗人的知己与酒伴，承载的是个人的孤独与自由。李白之月明亮而辽阔，充满仙气与动感。

- **杜甫写月** — 沉郁顿挫、忧国忧民。"露从今夜白，月是故乡明"——月是思乡与忧患的触发器。杜甫之月苍凉而厚重，映照的是家国之痛。

- **王维写月** — 空灵禅意、物我合一。"明月松间照，清泉石上流"——月是自然的一部分，无悲无喜。王维之月澄澈而宁静，体现的是禅宗式的观照。

- **张九龄写月** — 含蓄蕴藉、寄托深远。"海上生明月，天涯共此时"——月是人际情感的纽带。张九龄之月温润而辽远，承载的是对远方之人的思念。

### 阶段四：逐首深度鉴赏

对每首代表诗作，提供以下六层分析：

#### 4.1 原文与注释
```
诗文原文（忠实于《全唐诗》原文）

【注释】
- 关键字词读音与释义
- 典故出处
- 版本说明（如有异文需标注）
```

#### 4.2 白话译文
- 逐句白话对译
- 整体意境的白话传达
- 译文风格应传达原诗的韵味，避免机械直译

#### 4.3 逐句赏析
- 每句的意象选取、手法运用、情感表达
- 句与句之间的逻辑关系与情感递进
- 关键字词的炼字分析（"诗眼"所在）

#### 4.4 整体意境分析
- 诗歌营造的整体氛围与审美境界
- 虚实结合、动静相衬、情景交融等手法的运用
- 意境的层次：物境→情境→意境

#### 4.5 情感主旨
- 表达的核心情感（孤高 / 思乡 / 壮志 / 闲适 / 悲秋 / 伤别 / 怀古 / 闺怨等）
- 情感的层次与变化脉络
- 情感与诗人人生经历的关联

#### 4.6 艺术手法
- **比兴** — 以彼物比此物，先言他物以引起所咏之词
- **借景抒情** — 通过景物描写表达情感
- **托物言志** — 借具体事物表达志向品格
- **用典** — 引用历史故事或前人诗句
- **对仗** — 律诗中的工对、宽对、流水对
- **夸张 / 拟人 / 比喻 / 通感** — 修辞手法的运用
- **白描 / 工笔 / 写意** — 描写手法的差异

**鉴赏字数要求**：每首不少于500字，整体分析不少于2000字

### 阶段五：意境流派归类

将同一意象下的诗作按意境进行分类，帮助用户理解同一意象的丰富层次。

#### 梅花意象分类
- **傲雪凌霜型** — 孤高品格、不屈精神（王安石《梅花》："墙角数枝梅，凌寒独自开"）
- **暗香疏影型** — 清雅之美、幽独之姿（林逋《山园小梅》："疏影横斜水清浅，暗香浮动月黄昏"）
- **折梅寄远型** — 思乡怀人、寄托情思（陆凯《赠范晔》："折花逢驿使，寄与陇头人"）
- **梅花落型** — 伤春悲秋、美好消逝（李白《与史郎中钦听黄鹤楼上吹笛》："黄鹤楼中吹玉笛，江城五月落梅花"）

#### 竹意象分类
- **虚心有节型** — 谦虚品格、坚贞气节（白居易《画竹歌》）
- **幽篁独坐型** — 隐逸情趣、禅意境界（王维《竹里馆》："独坐幽篁里，弹琴复长啸"）
- **潇湘夜雨型** — 愁绪绵长、羁旅乡思（刘禹锡《潇湘神》）
- **翠竹生辉型** — 清新明丽、生机盎然（杜甫《严郑公宅同咏竹》）

#### 月意象分类
- **思乡怀人型** — 望月思乡、月夜怀人（李白《静夜思》："举头望明月，低头思故乡"）
- **禅意空灵型** — 月映禅心、空明澄澈（王维《鸟鸣涧》："月出惊山鸟，时鸣春涧中"）
- **历史沧桑型** — 月照古今、兴亡感慨（刘禹锡《石头城》："淮水东边旧时月，夜深还过女墙来"）
- **豪放飘逸型** — 举杯邀月、超然物外（李白《月下独酌》："举杯邀明月，对影成三人"）
- **闺怨相思型** — 月照空闺、思念远人（张九龄《望月怀远》："海上生明月，天涯共此时"）

#### 秋意象分类
- **悲秋感伤型** — 万物萧瑟、人生迟暮（杜甫《登高》："万里悲秋常作客，百年多病独登台"）
- **秋日壮阔型** — 天高气爽、壮志豪情（刘禹锡《秋词》："自古逢秋悲寂寥，我言秋日胜春朝"）
- **秋思羁旅型** — 秋风起处、乡愁万里（张继《枫桥夜泊》）
- **秋山隐逸型** — 秋山红叶、归隐之趣（王维《山居秋暝》）

### 阶段六：跨意象关联分析

分析意象之间的关联网络与组合规律。

#### 经典意象组合

- **梅+雪** — 同为冬日意象，常相互映衬。雪之白与梅之香对比，"梅须逊雪三分白，雪却输梅一段香"（卢梅坡《雪梅》）。两者组合强化了清冷高洁的意境。

- **月+酒** — 李白标志性组合，浪漫超脱。"花间一壶酒，独酌无相亲。举杯邀明月，对影成三人"（李白《月下独酌》）。月与酒的组合构成了唐诗中最经典的孤独美学。

- **秋+雁** — 典型悲秋意象群。雁为候鸟，秋天南飞，天然承载离别与思乡之情。"乡书何处达？归雁洛阳边"（王湾《次北固山下》）。

- **柳+送别** — 折柳赠别传统，"柳"谐音"留"。"渭城朝雨浥轻尘，客舍青青柳色新"（王维《送元二使安西》）。

- **竹+隐逸** — 王维式隐逸情趣。"独坐幽篁里，弹琴复长啸"（王维《竹里馆》）。竹与琴、月的组合营造出世外高人的意境。

- **松+坚贞** — 岁寒三友（松竹梅），品格象征。"岂不罹凝寒，松柏有本性"（刘桢《赠从弟》）。

- **山水+闲适** — 王维、孟浩然的标志意象群。"空山新雨后，天气晚来秋"（王维《山居秋暝》）。

- **边塞+壮志** — 高适、岑参的标志意象群。大漠、烽火、黄沙、玉门关等构成壮阔的边塞意境。

#### 意象组合规律
- **季节意象+情感意象**：秋+悲、春+喜、冬+孤、夏+闲
- **地理意象+空间意境**：山+幽深、江+辽阔、海+壮阔、溪+清幽
- **动物意象+人格象征**：雁+思乡、鹤+高洁、蝉+清高、马+壮志
- **器物意象+生活场景**：酒+豪情、琴+雅趣、剑+侠气、舟+漂泊

### 阶段七：学习路径推荐

根据用户兴趣和水平，推荐三级递进学习路径。

#### 入门级：最经典的10首意象诗
每个意象选取1-2首最经典、最广为传诵的作品：

- **梅花**：王安石《梅花》→ 陆游《卜算子·咏梅》
- **竹**：王维《竹里馆》→ 郑燮《竹石》
- **月**：李白《静夜思》→ 张九龄《望月怀远》
- **山**：王维《山居秋暝》→ 杜甫《望岳》
- **水**：李白《望庐山瀑布》→ 刘禹锡《浪淘沙》
- **送别**：王维《送元二使安西》→ 李白《赠汪伦》
- **思乡**：李白《静夜思》→ 王维《九月九日忆山东兄弟》
- **秋**：杜甫《登高》→ 刘禹锡《秋词》
- **边塞**：王昌龄《出塞》→ 王之涣《凉州词》
- **田园**：孟浩然《过故人庄》→ 王维《渭川田家》

#### 进阶级：风格对比与意象群分析
- 同一意象在不同诗人笔下的风格差异（如李白之月 vs 杜甫之月 vs 王维之月）
- 同一诗人不同人生阶段的意象运用变化（如李白青年时的月 vs 流放后的月）
- 同一意象在不同体裁中的表现差异（五言绝句 vs 七言律诗 vs 古体诗）

#### 深度级：意象演变史
- 从初唐到晚唐同一意象的文化内涵演变
- 唐诗意象对后世（宋词、元曲、明清小说）的影响
- 意象的原型追溯（《诗经》《楚辞》中的原始意象→唐诗中的成熟意象）
- 中外文学意象比较（如中国之月 vs 西方文学中的moon意象）

## 输出格式规范

### 标准输出结构（单个意象查询）

```
一、<意象>意象概述（200字）
    — 该意象在唐诗中的总体地位、文化内涵、常见情感寄托

二、代表诗人（每人100-150字）
    — 生平简介、该意象偏好、创作风格

三、代表诗作鉴赏（每首500-800字）
    — 原文 → 注释 → 译文 → 逐句赏析 → 意境 → 手法

四、意境流派归类（300字）
    — 同一意象下的不同意境类型

五、跨意象关联（200字）
    — 与该意象常搭配的其他意象

六、学习路径推荐（150字）
    — 入门→进阶→深度三级建议

七、延伸探索（3个问题）
    — 引导用户继续深入的问题
```

### 格式约束
- **语言**：中文输出，诗文保留原貌
- **字数**：每首鉴赏≥500字，整体分析≥2000字
- **不用表格**：所有对比和分析用段落或项目符号列表呈现
- **来源标注**：关键引用标注出处（如"《全唐诗》卷XXX"）
- **版本说明**：如有繁简差异或版本异文需注明

## 执行规则

### 检索规则
1. 每次查询至少执行4轮搜索，覆盖古诗文网、学术资源、权威数据库
2. 诗文引用必须忠实于《全唐诗》原文，不可杜撰或拼凑
3. 无法确认的诗文标注"此诗出处待查证"
4. 学术争议需注明不同说法

### 鉴赏规则
1. 兼顾学术严谨性与通俗可读性
2. 避免过度学术化，让古典文学初学者也能理解
3. 每首鉴赏须有独到见解，避免人云亦云
4. 艺术手法分析须结合具体诗句，不可空谈

### 交互规则
1. 支持主动提问引导用户深入探索
2. 每次鉴赏结束后，提供2-3个延伸问题供用户选择
3. 根据用户反馈动态调整分析深度和侧重点
4. 初学者可侧重基础赏析，进阶者可侧重风格比较和意象演变

## 注意事项

- **文献忠实性** — 所有诗文引用必须基于《全唐诗》原文，版本差异需标注
- **学术严谨性** — 分析有据可查，避免主观臆断，学术争议需注明
- **通俗可读性** — 避免过度学术化，让普通爱好者也能理解并享受
- **互动引导性** — 主动提问引导深入，而非被动等待指令
- **意象完整性** — 覆盖不同意象类别，避免偏重某一类
- **持续迭代** — 根据用户反馈不断优化鉴赏深度和表达方式

## Pitfalls & 实战经验

基于首次实战（"松"意象查询），总结以下经验：

1. **诗人生平不可跨朝代混淆** — 搜索结果常将魏晋诗人（如刘桢）混入唐代。检索时必须核实诗人的生卒年和朝代归属。刘桢是建安七子，非唐人，但其《赠从弟》是松意象的源头之作，应作为"前唐·文化源头"收录，需明确标注朝代。

2. **搜索结果中古诗文网站的OCR/编码错误较多** — 如"gushiwen.cn"的赏析文本中常出现"体松"替代"孤松"、"叶云"替代"凌云"等乱码。引用时必须交叉比对权威底本（如中华书局版《全唐诗》），不可直接照抄网站赏析文本。

3. **每首鉴赏500字+整体2000字的要求在单次输出中容易超长** — 当意象涉及5位以上诗人、8首以上诗作时，总输出可达8000-15000字。建议对用户说明输出较长，分段呈现，方便阅读。

4. **"同题材风格差异对比"是最受用户欢迎的维度** — 如"李白之松 vs 王维之松 vs 刘桢之松"的对比分析，比单独鉴赏每首诗更能揭示意象的丰富层次。应优先展开此维度。

5. **跨意象关联需要"有据可查"** — 不能凭主观想象编造意象组合规律，需在具体诗作中找到实例支撑。如"松+风"组合有王维"松风吹解带"、刘桢"瑟瑟谷中风"等实例。

6. **意境流派归类应有清晰的命名和代表作** — 每种意境类型需有明确的名称（如"岁寒坚贞型""凌云壮志型"）、核心精神概括、和1-2首代表作及名句。
