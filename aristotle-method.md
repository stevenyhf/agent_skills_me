## 二、核心世界观（亚里士多德的认知底层）

### 2.1 经验主义：真理在感官世界

**核心命题**：
- 与柏拉图"真理在另一个世界"相反，亚里士多德坚持真理就在感官世界中
- 知识始于经验观察，但不限于经验——通过归纳上升到普遍原理
- "每当理论与观察相矛盾时，必须信任观察"

**认知路径的五层阶梯**：
1. **感知**（aisthesis）— 直接接触具体事物
2. **记忆**（mneme）— 保存感知内容
3. **经验**（empeiria）— 多次感知和记忆积累
4. **技艺**（techne）— 从经验中提炼普遍规则
5. **科学**（episteme）— 理解事物的原因（aitia）

**关键洞察**：
- 这是科学方法论的先驱
- 从"看到"到"理解"需要跨越五层——大多数人在第3层（经验）就停了
- 亚里士多德自己就是这种方法的典范——他描述了500多个动物物种

### 2.2 万物皆有目的（Telos）

**核心命题**：
- 每个存在的事物都有一个"目的"（telos）——它的存在是为了什么？
- 橡子的telos是成为橡树；种子的telos是成为植物；人的telos是过理性生活
- 理解事物的telos是理解事物本身的关键

**关键洞察**：
- 目的论不是迷信——它是理解功能、设计和意义的有力工具
- 在生物学中，器官的telos就是它的功能
- 在组织中，团队的telos就是它存在的使命
- 在个人中，你的telos就是你作为"人"的卓越功能的实现

### 2.3 人是什么——功能论证（Ergon Argument）

**核心命题**：
- 要知道什么是"好的人"，先要知道"人的功能"（ergon）
- 刀的功能是切割——好的刀就是切割得好
- 人的独特功能是"理性活动"——好的人就是理性活动得好
- 因此幸福（eudaimonia）= "灵魂与卓越美德相契合的理性活动"

## 三、底层思维框架（六大支柱）

### 框架 1：四因说 — 全面因果分析

**核心命题**：
- 要真正理解任何事物，必须回答四个不同的"为什么"
- 大多数现代分析只关注1-2个因（尤其是动力因），忽略了其他——这是严重的认知盲区

**四因详解**：
- **质料因**（Material Cause）：由什么构成？（物质基础/资源/人员组成）
- **形式因**（Formal Cause）：结构/模式是什么？（设计/架构/规则/理念）
- **动力因**（Efficient Cause）：谁/什么使其产生？（创造者/推动者/触发事件）
- **目的因**（Final Cause / Telos）：为什么存在？目的是什么？（使命/功能/最终目标）

**应用 Prompt Template**：
```
用四因说全面分析 [事物/现象/问题]：

1. **质料因**：它由什么构成？________
2. **形式因**：它的结构/组织方式是什么？________
3. **动力因**：谁或什么导致了它的存在？________
4. **目的因（Telos）**：它存在的目的是什么？________

完整性检查：
5. 四因中你忽略了哪个？________（最常见的是忽略目的因）
6. 如果补上缺失的因，你的理解会怎样变化？________
```

### 框架 2：中庸之道 — 决策校准

**核心命题**：
- 美德不是极端，是两个极端之间的"适中"
- 但"适中"不是数学平均——它是因人、因时、因情境而异的最优点
- 需要"实践智慧"（Phronesis）来判断什么是当前的"适中点"

**经典中庸对照**：
- 勇敢 = 鲁莽（过度）— **勇敢（适中）**— 怯懦（不足）
- 节制 = 放纵（过度）— **节制（适中）**— 麻木（不足）
- 慷慨 = 挥霍（过度）— **慷慨（适中）**— 吝啬（不足）
- 大度 = 虚荣（过度）— **大度（适中）**— 自卑（不足）
- 温和 = 暴躁（过度）— **温和（适中）**— 麻木不仁（不足）
- 友好 = 谄媚（过度）— **友好（适中）**— 冷漠（不足）
- 真诚 = 自夸（过度）— **真诚（适中）**— 自贬（不足）
- 勤奋 = 过度工作（过度）— **勤奋（适中）**— 懒惰（不足）

**应用 Prompt Template**：
```
用中庸之道校准 [你的决策/行为/策略]：

1. 你当前面临的选择是什么？________
2. 这个领域的"过度"状态是什么？________
3. 这个领域的"不足"状态是什么？________
4. 你目前偏向哪个极端？________
5. 对你而言、在当前情境下，"适中点"大约在哪里？________
6. 向适中点调整需要什么具体行动？________

注意："适中"不是算术平均，而是需要"实践智慧"来定位。
咨询有经验的导师/前辈可以帮助你找到更准确的适中点。
```

### 框架 3：幸福论（Eudaimonia）— 人生终极设计

**核心命题**：
- 人生的最高目标不是快乐（hedonia），而是幸福（eudaimonia）
- 幸福 = 人的功能（理性活动）的卓越实现
- 幸福需要：美德 + 理性活动 + 持续一生 + 适度外在条件

**幸福论完整框架**：
1. 什么是你作为"人"的独特功能？（你的理性活动是什么？）
2. 你在哪些方面可以做到"卓越"（aretê）？
3. 你的卓越是否需要"持续一生"（一只燕子不能构成春天）
4. 你是否具备适度的外在条件？（健康、友谊、适度财富）

**应用 Prompt Template**：
```
用幸福论重新设计你的人生方向：

1. **你的功能**：你最卓越的理性活动是什么？（写作/分析/教导/创造/治愈/领导？）________
2. **你的美德**：你正在培养哪些卓越品质？________
3. **持续性**：你的人生计划能持续30年吗？还是一场短跑？________
4. **外在条件**：你的健康、友谊、经济基础是否充足？________
5. **反思**：如果幸福不是"感觉好"而是"活得好"，你当前的方向需要怎么调整？________
```

### 框架 4：三段论 — 严谨推理

**核心命题**：
- 推理的有效性取决于形式，不取决于内容
- 三段论：大前提 → 小前提 → 结论
- 如果前提为真且形式有效，结论必然为真

**经典三段论**：
- 大前提：所有人都是会死的
- 小前提：苏格拉底是人
- 结论：苏格拉底是会死的

**常见谬误检查**：
- 肯定后件谬误：如果P则Q，Q为真，所以P为真（错！）
- 否定前件谬误：如果P则Q，P为假，所以Q为假（错！）
- 中项不周延：大前提和小前提的"中项"没有在至少一个前提中全称使用
- 四词谬误：三段论中出现了四个不同的词（概念混淆）

**应用 Prompt Template**：
```
用三段论构建严谨论证 [你的主张]：

1. **结论**：你想证明什么？________
2. **大前提**：支撑结论的一般性原则是什么？________
3. **小前提**：具体情况/事实是什么？________
4. **形式检验**：从大前提+小前提，结论是否必然成立？________
5. **前提真伪**：大前提和小前提是否都为真？________
6. **谬误检查**：是否存在上述任何谬误？________
```

### 框架 5：形式-质料统一论 — 结构与材料分析

**核心命题**：
- 每个事物都是"质料"（matter）和"形式"（form）的统一体
- 质料 = 构成事物的"原料"；形式 = 事物的"内在结构"和"本质"
- 二者不可分割——没有无形式的质料，没有无质料的形式（除了"不动之推动者"——上帝/纯粹形式）

**层次结构**：
- 砖头：泥土（质料）+ 砖的形式
- 墙壁：砖头（质料）+ 墙壁的形式
- 房屋：墙壁（质料）+ 房屋的形式
- 每一层的"形式"可以成为下一层的"质料"

**应用 Prompt Template**：
```
用形式-质料分析 [你的产品/项目/团队]：

1. **质料**：它由什么构成？（资源/人员/数据/材料）________
2. **形式**：它的内在结构是什么？（架构/流程/设计）________
3. **统一性**：质料和形式是否匹配？________
   - 有好材料但没好结构？→ 需要重新设计
   - 有好设计但没好材料？→ 需要获取更好的资源
4. **层次定位**：这个事物在更大系统中是"质料"还是"形式"？________
5. **目的因**：这个形式是为了实现什么目的而存在？________
```

### 框架 6：实践智慧（Phronesis）— 情境判断力

**核心命题**：
- 实践智慧（Phronesis）不是书本知识，而是在具体情境中做出正确判断的能力
- 它是通过经验、反思、指导逐步培养的能力
- 它是中庸之道能够运作的"前提条件"——没有它，你不知道"适中点"在哪里

**Phronesis vs. Sophia vs. Techne**：
- **Sophia（理论智慧）**：关于永恒真理的知识——"世界是什么"
- **Techne（技艺）**：关于如何制作东西的知识——"怎么做"
- **Phronesis（实践智慧）**：关于在具体情境中什么是正确行动的知识——"该做什么"

**培养Phronesis的路径**：
1. 积累经验——在不同情境中做决策
2. 反思决策的结果
3. 向有实践智慧的人学习（导师/前辈）
4. 研究过去的案例（历史是最好的老师）
5. 在具体情境中练习判断

**应用 Prompt Template**：
```
用实践智慧分析 [你面临的具体情境]：

1. **情境描述**：具体发生了什么？谁参与？什么时间？什么限制？________
2. **相关先例**：过去类似的情境中，最好的结果是怎么产生的？________
3. **导师视角**：如果你的导师面对这个情境，会怎么做？________
4. **风险考量**：最可能出错的是什么？________
5. **行动选择**：在这些考量下，你认为最"明智"（而非"正确"）的行动是什么？________
6. **反思框架**：行动之后如何复盘？________
```

## 四、落地执行方法论（七大模块）

### 模块 1：四因分析法 — 全面诊断

**场景**：分析某个产品、组织、现象或问题

**亚里士多德式方法**：
1. 质料因：它由什么资源/材料/人员构成？
2. 形式因：它的结构/设计/流程是什么？
3. 动力因：谁创造了它？什么事件触发？
4. 目的因：它存在的目的是什么？
5. 完整性检查：四因是否全部被考虑？
6. 改进方向：哪个因是当前的短板？

### 模块 2：中庸校准法 — 纠偏决策

**场景**：你/你的团队在某方面明显偏向一个极端

**亚里士多德式方法**：
1. 明确：你在哪个"光谱"上做决策？
2. 识别两个极端
3. 评估你当前在光谱上的位置
4. 寻找适中点（需要实践智慧！）
5. 制定微调方案
6. 持续监测（适中点不是固定的，会随情境变化）

### 模块 3：Eudaimonia审计法 — 人生评估

**场景**：每季度/每年审视人生方向

**亚里士多德式方法**：
1. 我的"功能"（ergon）是什么？我是否在发挥它？
2. 我培养的美德是否足够？哪些需要加强？
3. 我的生活是否"可持续30年"？还是只看到明年？
4. 外在条件（健康、友谊、经济）是否需要关注？
5. 过去的一个季度，我是"感觉好"还是"活得好"？

### 模块 4：三段论论证法 — 构建说服力

**场景**：你需要说服他人或验证自己的推理

**亚里士多德式方法**：
1. 明确你的结论
2. 倒推：什么大前提+小前提可以支撑这个结论？
3. 验证大前提（是否普遍为真？）
4. 验证小前提（是否事实为真？）
5. 检查形式（逻辑是否有效？）
6. 预测反对意见并预先反驳

### 模块 5：形式-质料优化法 — 系统改善

**场景**：你的产品/组织/项目存在结构性问题

**亚里士多德式方法**：
1. 质料审计：当前"材料"（人员/资源/技术）是否足够？
2. 形式审计：当前"结构"（流程/设计/制度）是否合理？
3. 匹配分析：质料和形式是否协调？
4. 层次检查：在更大系统中，你的"形式"是否需要升级？
5. 目的因对齐：你的形式是否服务于正确的目的因？

### 模块 6：美德培养法 — 品格发展

**场景**：你需要培养某项品格/美德

**亚里士多德式方法**（道德美德靠习惯养成）：
1. 选择你要培养的美德（如勇气、慷慨、温和等）
2. 定义这个美德在你情境中的"适中"表现
3. 识别你通常偏向的极端（过度还是不足）
4. 制定"反极端"的练习计划——刻意走向另一个方向
5. 记录你的实践——每周检查是否在向适中点移动
6. 持续至少3个月——美德靠习惯形成，习惯需要时间

### 模块 7：实践智慧积累法 — 判断力训练

**场景**：你面对一个需要"智慧"而非"知识"的复杂决策

**亚里士多德式方法**：
1. 这不是一个可以靠公式解决的问题——承认情境的复杂性
2. 列出相关的先例/案例（历史的、个人的、朋友的）
3. 咨询你尊敬的导师/前辈（他们的Phronesis可能比你丰富）
4. 考量行动的长远后果（不仅仅是当下）
5. 在多个可行方案中选择最"明智"而非最"正确"的
6. 行动后复盘：结果如何？什么可以改进？
