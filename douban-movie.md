# 豆瓣电影查询与影评分析

## 功能概述

通过豆瓣电影页面和第三方API获取电影信息，生成包含评分、演职员、剧情、
短评摘要、影评分析的综合报告。内置《浩峰看电影》风格的深度影评写作能力。

## 数据源（按优先级）

### 1. web_extract 直接抓取豆瓣页面（首选）

- 豆瓣电影详情页可通过 web_extract 直接访问，无需登录
- URL格式：`https://movie.douban.com/subject/{id}/`
- 可获取：片名、评分、评分分布、演职员、奖项、热门短评标题和票数
- 局限：剧情简介可能被"展开"按钮截断；评论正文需要JS渲染

### 2. api.wmdb.tv 第三方API（结构化补充）

- 免费、无需认证
- 搜索：`https://api.wmdb.tv/api/v1/movie/search?q={keyword}&limit=5`
- 详情：`https://api.wmdb.tv/movie/api?id={douban_id}`
- 返回：IMDB评分、烂番茄评分、演员表、剧情简介等

### 3. web_search 搜索豆瓣链接

- 当用户只提供片名（无URL）时，用搜索找到豆瓣页面
- 查询格式：`"{片名}" site:movie.douban.com`

## 执行工作流

### 步骤1：获取电影URL

- 用户提供URL → 直接使用
- 用户提供片名 → web_search `"片名" site:movie.douban.com` 找到URL
- 提取 douban subject ID（URL中的数字）

### 步骤2：并行获取数据

同时调用：

1. `web_extract` 抓取豆瓣电影详情页
2. `web_extract(urls=[wmdb_url])` 从wmdb.tv获取补充数据（如有douban_id）

### 步骤3：并行获取扩展数据

同时调用以下数据源，用于新增的四大分析板块：

1. **时代背景**：`web_search` 搜索 `"{片名}" 导演 拍摄背景 创作动机 幕后`
2. **社会影响**：`web_search` 搜索 `"{片名}" 社会影响 文化现象 讨论`
3. **多平台评论**：用英文名搜索国际平台（wmdb.tv返回的originalName
   或alias字段含英文名）。搜索示例：
   `"{英文名}" {年份} {导演英文名} review OR Letterboxd OR IMDB`
   额外平台：Letterboxd（文艺片/影迷社区，评论质量高）、
   Jonathan Rosenbaum网站（美国著名影评人，华语片评论权威）
   搜索：`"{英文名}" site:letterboxd.com` 和 `"{英文名}" site:jonathanrosenbaum.net`
4. **精选豆瓣评论**：
   a. 先用 `web_search` 搜索 `"片名" 豆瓣影评 site:movie.douban.com/review`
   b. 从搜索结果snippet中查看各篇"有用"数，选出票数最高的3篇
   c. 用 `web_extract` 批量抓取这3篇影评详情页（`/review/{id}/`）
   d. 豆瓣影评详情页（/review/{id}/）可稳定抓取，含完整影评全文
   e. 注意区分同名片影评（如"似水流年"有1984电影和2003电视剧两个条目）

### 步骤4：解析与整合

从豆瓣页面提取：

- 片名（中英文）
- 年份、导演、编剧、主演
- 评分、评分分布
- 类型、制片国家/地区、语言、片长
- 奖项信息
- 热门短评（标题+有用数）
- 热门影评（标题+链接ID）

从wmdb.tv补充：

- 剧情简介（豆瓣页面可能截断）
- IMDB评分、烂番茄评分（如有）

从web_search补充：

- 导演创作背景、拍摄动机
- 社会文化影响报道
- 多平台评分与评论摘要

### 步骤5：生成分析报告

按以下结构输出（中文）：

```
标题：《{片名}》深度解析

一、基本信息
  片名、年份、导演、编剧、主演、类型、国家、语言、片长

二、评分分析
  豆瓣评分及分布、IMDB评分（如有）、评分解读

三、剧情概述
  来自wmdb或页面提取的剧情简介

四、主创团队分析
  导演风格、主要演员、创作背景

五、荣誉与影响
  获奖情况、行业影响

六、拍摄背景与时代文化因素
  该片拍摄时的时代背景、社会语境、文化潮流，
  导演的创作动机是什么——什么事件、情绪或思潮促使导演拍这部电影？
  引用导演访谈、幕后花絮、新闻报道中的原始素材。

七、当代社会与文化影响
  这部电影对当下社会、文化产生了哪些影响？
  是否引发了公共讨论、改变了流行文化、催生了亚文化现象？
  影片的主题在今天是否仍有现实意义？

八、多平台评论汇总
  来自IMDB、烂番茄（Rotten Tomatoes）、Metacritic、
  Letterboxd（影迷社区）、Jonathan Rosenbaum（权威影评人）等
  平台的评分和核心评价观点。
  各平台之间的评分差异及其可能的原因分析。

九、豆瓣精选用户评论深度分析
  选取豆瓣用户评论中最优秀的3篇（按"有用"数排序），
  对每篇进行深度总结：
  - 评论者核心观点
  - 评论者的分析角度和论证逻辑
  - 最有价值的洞察
  - 三篇评论之间的共识与分歧
  最后综合提炼用户评论中的集体智慧。

十、观众反馈
  热门短评摘要、观众关注焦点

十一、综合评价
  基于以上全部信息的独立分析与推荐
```

## 影评写作风格（浩峰看电影风格）

### 核心定位

深度挖掘影视作品的人文内核，串联创作背景与当代社会议题，
既还原作品的创作语境，也分析其对当下的借鉴意义，
拒绝流量化的肤浅解读，坚持专业、客观的影视评析逻辑。

### 写作风格五维度

| 维度           | 表现                                                             |
| ------------ | -------------------------------------------------------------- |
| **语气**       | 自信果断，敢说敢言，不暧昧、不敷衍。批评时直截了当（"难看，垃圾，快跑"），赞美时也不吝啬（"好看，好看，还是TM的好看"） |
| **语言层次**     | 同时驾驭多种语体：可诗意克制、可犀利吐槽、可学术分析、可口语吐槽。不同篇章的语体取决于作品本身的气质             |
| **修辞特点**     | 善用类比和隐喻将电影语言翻译成人人能理解的感受；善用具象意象承载抽象情感（如"冰下的阳光"形容《杨之后》）          |
| **人物代入感**    | 有强烈的"我"在场的意识。观影是带有体温的私人行为，影评允许包含个人回忆、青春叙事和情感投射                 |
| **网络语言适度使用** | 适当使用口语/网络语感，不端着，不做作；但分寸感好，从不用网络语言破坏严肃话题的庄重                     |

### 覆盖内容范围

已收录栏目覆盖的影视类型及代表作品：

1. **历史题材短剧**：《叶卡捷琳娜二世》（俄罗斯）—— 分析沙俄时期的政治生态与女性掌权的历史逻辑
2. **经典现实题材影片**：《小鞋子》（伊朗）—— 解读底层视角下的尊严与亲情叙事
3. **战争反思题材**：《穿条纹睡衣的男孩》—— 剖析集中营题材的儿童视角表达与反战内核
4. **深度纪录作品**：《海豚湾》—— 梳理环保议题的纪录手法与公共影响
5. **科幻影视IP**：《电子梦，菲利普·迪克的世界》—— 拆解赛博朋克题材的哲学思考与科幻落地

## 影评方法论

### 核心分析路径（6步法）

**第一步：还原基本信息**——先说清楚影片的上映时间、导演、主演、类型、地区、片长等事实信息，确保无硬伤。

**第二步：找准影片的核心定位**——这部作品想做什么、服务的观众是谁？它的野心和边界各在哪里？

**第三步：拆解执行层面**——对叙事结构、视听语言、表演、剪辑节奏、剧本逻辑逐项检验，结合具体场景举例说明优缺点，不说空话。

**第四步：判断"意图 vs 完成度"**——这部作品想要完成的目标是否真正达成了？野心与能力是否匹配？

**第五步：社会文化解读**——将影片情节、人物关系和价值取向链接到当下社会议题（代际、性别、技术、城市化等），给出有纵深的分析。

**第六步：给出结论**——明确说出评分（即使是有条件推荐）、该不该看、对哪类观众友好/不友好。

## 评价标准体系

### 六维评判体系

| 维度          | 权重  | 核心问题                                     |
| ----------- | --- | ---------------------------------------- |
| **剧本与结构**   | 较高  | 叙事逻辑是否自洽？情节节点是否莫名其妙？人物动机是否有说服力？          |
| **表演与角色**   | 较高  | 演员是否在角色里？角色是否服务于演员还是被演员塑造？有没有演员被剧本拖累的问题？ |
| **视听语言**    | 中等  | 摄影、剪辑、配乐是否贴合影片气质？导演的视觉风格是否有辨识度？          |
| **类型完成度**   | 较高  | 商业片要完成类型承诺，动作片就要打得爽，喜剧片就要好笑。承诺是否兑现？      |
| **价值观与思想性** | 中等  | 影片的价值观是进步还是退步？有没有隐性反智叙事？代际权力的表达是否粗暴？     |
| **原创性**     | 低   | 在多大程度上超出了同类型已有作品？有没有照搬套路而不自知的问题？         |

### 附加参考维度

- **票房/评分对比**：Douban分数是否虚高？评分与实际观感是否匹配？警惕"报复性好评"
- **导演履历纵向对比**：本片与导演前作相比，是进步还是退步？风格是否有传承/突破？
- **演员发展曲线**：该演员在这部作品里的表演是其舒适区还是有所突破？
- **代际元素**：新生代演员是否真的能接班？老一代的经验叙事是否在影片中被过度美化？
- **续作/IP连贯性**：IP电影是否在吃老本？情怀是有效的还是过时的？

### 底线原则

- **反情怀崇拜**：拒绝无脑吹捧导演或演员的历史地位；过往成就是过往的，眼前的评分只看眼前这部作品
- **拒绝平庸宽容**：烂片直言烂，不以"有诚意""情怀"为烂片辩护；平庸不是中性词
- **具象化批评**：指出具体哪个场景、哪个表演、哪个剪辑出了问题，不停留于"不好看""节奏差"
- **同理心评分**：同一评分对不同类型的电影意义不同，动作片的7分不等于文艺片的7分

## 写影评时的回答原则

1. **先还原作品基础信息**：上映时间、主创团队、核心剧情，避免信息错误
2. **挖掘作品的独特创作手法**：比如《穿条纹睡衣的男孩》的儿童视角创新，《海豚湾》的隐秘拍摄手法
3. **关联当代议题**：将作品的核心矛盾与当下的社会、文化、技术议题结合，输出有延续性的分析
4. **保持中立客观**：不输出极端化的评价，呈现作品的争议性与多元解读空间
5. **控制篇幅**：单部作品的解读控制在3000字以内，结构清晰，重点突出，符合栏目的内容节奏

## 注意事项

1. **反爬机制（实测2026-05-30）**：
   
   - 豆瓣详情页有时返回"载入中..."（JS动态渲染未完成），不可靠
   - 豆瓣影评详情页（/review/{id}/）可稳定抓取，含完整影评全文
   - wmdb.tv API始终可用，是结构化数据的可靠来源
   - 降级策略：详情页失败时，用wmdb.tv API（含剧情/评分/演员）+
     web_search结果中的描述片段 + 影评页面组合生成报告

2. **数据完整性**：web_extract返回的是页面摘要（约5000字符），
   超长页面会被截断。优先使用关键信息字段。

3. **用户偏好**：用户偏好简洁中文回复，输出到本地文件时使用
   `~/` 目录，临时文件用 `/tmp/`。

4. **影评风格选择**：
   
   - 查询报告：使用标准分析报告结构（步骤5）
   - 深度影评：使用"浩峰看电影"风格（六维评价体系 + 6步法）
   - 简短评价：直接给出评分和核心观点

5. **批量查询**：多次查询时注意间隔（建议>2秒），避免触发
   豆瓣频率限制。

6. **国际评论搜索技巧**：搜索国际平台评论时，必须使用影片的英文名
   （如"Homecoming"而非"似水流年"），英文名可从wmdb.tv API的
   `originalName`或`alias`字段获取。加上导演英文名和年份可提高精度。

7. **同名片辨析**：豆瓣存在同名不同年代的影片（如"似水流年"有1984年
   电影和2003电视剧），搜索影评时需确认review所属的subject ID，
   避免张冠李戴。

8. **跨片对比分析**：在"拍摄背景"板块中，引入同年份或同题材的影片
   进行对比可大幅提升分析深度（如《似水流年》vs《省港旗兵》，
   一个是香港人回内地寻根，一个是内地人到香港闯荡）。
   搜索：`"{片名}" 同年 对比 OR 并置`

## 可用脚本

> 搜索查询模板和数据源可靠性排名见 `references/search-queries.md`

### 搜索并获取电影信息（execute_code示例）

```python
from hermes_tools import web_search, web_extract

# Step 1: 搜索
results = web_search(query='"肖申克的救赎" site:movie.douban.com', limit=3)
url = results['data']['web'][0]['url']  # 取第一个结果

# Step 2: 提取详情
page = web_extract(urls=[url])
print(page)
```

### wmdb.tv API 补充数据

```python
from hermes_tools import web_extract

# 搜索（按关键词）
search_url = "https://api.wmdb.tv/api/v1/movie/search?q=肖申克的救赎&limit=3"
# 详情（按douban ID）
detail_url = "https://api.wmdb.tv/movie/api?id=1292052"
result = web_extract(urls=[detail_url])
```
